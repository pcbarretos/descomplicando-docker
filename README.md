# Descomplicando Docker
Conteúdo Abordado na Apresentação.

## Instalando Docker
- O Docker possui as seguintes formas de instalação:

  - Windows
    - *https://docs.docker.com/docker-for-windows/install/*
  - MacOS
    - *https://docs.docker.com/docker-for-mac/install/*
  - Linux
    - *https://docs.docker.com/engine/install/*
- E importante vericar os requisitos mínimos para instalação correta em seu Sistema Operacional.

## Conhecendo Docker Hub
- Acesse: *https://hub.docker.com/* e crie a sua conta, assim você podera criar repositórios e subir as imagens criadas por vocês.
  * Aqui voce podera procurar por diversas imagem tanto oficiais quanto criadas por outros usuários e pela comunidade.

### Executando Primeiro Contêiner
- Uma vez que já instalamos o Docker e temos a nossa conta criada, já podemos testar o docker para saber se está tudo certo! Vamos lá.
- Abra um terminal ou Power Shell e digite:
  <br></br>
  * `docker run hello-wolrd`
  <br><br>
  
- Se voce seguiu todo o passo a passo de instalação de acordo com o seu Sistema Operacional corretamente, esse será o retorno no seu teminal/cmd/PowerShell.
  <br> </br>
  ![dockerhub](https://4sysops.com/wp-content/uploads/2018/01/Running-the-docker-hello-world-image.png)
  <br> </br>
  - Se retornou como mostrado acima, Parabens! seu Docker está pronto pra usar.
